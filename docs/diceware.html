Generate secure passwords

Download and run diceware:

$ curl -O https://www.romanzolotarev.com/bin/diceware
$ . diceware
uerrilla agnostic backdoor glove jealous mummy dictionary sloth

On every run it generates a random 8-word passphrase. Write this phrase down or just memorize it.
